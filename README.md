# US Agricultural Production Analysis

## 📌 Project Overview
This project analyzes agricultural production data from the USDA across multiple states and years.

## 🎯 Objectives
- Analyze production trends by commodity
- Identify high-performing states
- Detect anomalies and gaps in data

## 🗂️ Dataset
- milk_production
- cheese_production
- coffee_production
- honey_production
- yogurt_production
- state_lookup

## 🛠️ Tools Used
- SQL (DBeaver)
- Power BI
- Git & GitHub

## 📊 Key Insights
- California and Wisconsin lead in dairy production
- Coffee production shows fluctuations across years
- Some states report missing production values
